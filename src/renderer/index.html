📖 Standalone MCP 接入配置（TRAE Work / Claude Desktop） ▶
方式一：项目级配置（推荐）
项目根目录已包含 .mcp.json 文件，TRAE Work 打开本项目时会自动发现并加载 MCP 服务。
方式二：手动配置
将以下 JSON 添加到 TRAE Work 的 MCP 配置中（设置 → MCP 服务器 → 添加）：
{ "mcpServers": { "web-scout": { "command": "node", "args": ["<项目路径>/src/main/mcp-standalone.js"], "env": { "MCP_READONLY": "false" } } } }
💡 提示：standalone 模式无需启动 Electron 应用，可直接读取已保存的工作流/卡片数据。
若需真实抓取 SPA 站点（B站/抖音等），仍需启动本应用并开启上方"MCP 服务状态"。
