# WebScout - 可视化智能网页资源提取器

![version](https://img.shields.io/badge/version-2.0.1-blue)
![platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux-lightgrey)
![license](https://img.shields.io/badge/license-MIT-green)

一款基于 Electron 的跨平台桌面应用，在普通浏览器的基础上，额外提供一套**可视化抓取模式**，让用户可以像浏览网页一样简单地获取图片、音视频、文档和文本内容。v2.0.1 升级**HT 编辑器专业化系统**与**展览编辑器**，让数据可视化与演示展示的链路一键打通。

---

## v2.0.1 重大升级亮点

- **HT 编辑器专业化升级**：菜单栏重构为 4 组（文件/编辑/插入/视图），新增撤销/重做系统、多选对齐工具（8 种对齐模式）、5 种新卡片类型（仪表盘/统计卡/对比卡/漏斗图/流程图）
- **展览编辑器**：自定义卡片框布局，支持添加框→调整→双击选卡→选展示方式，4 种展示方式（适应/填充/拉伸/居中），配置持久化到文档
- **跨平台 CI 分离**：Windows 与 Linux 安装包分别构建，用户从 GitHub Releases 按需下载，无需全部拉取
- **卡片布局优化**：flex 布局修复，内容超出自动滚动；色条系统扩展到 22 种卡片类型
- **新增 3 个文档模板**：数据仪表盘、AI 工作流编排、项目复盘

详细变更见 [CHANGELOG.md](./CHANGELOG.md)。

## 项目简介

**是什么：** 一款轻量级跨平台桌面应用（Electron），内置浏览器 + 可视化资源提取引擎 + HT 编辑器 + AI 工作流。

**面向谁：** 内容创作者、研究人员、自媒体运营者、HR、电商运营、学生群体——任何需要从网页中批量获取多媒体资源并进行统计分析的人。

**核心功能：**

| 功能 | 说明 |
|------|------|
| 双模式切换 | 默认浏览模式（与普通浏览器无异），一键开启抓取模式 |
| 资源自动识别 | 自动扫描页面中的图片、视频、音频、PDF文档和文本内容 |
| 可视化操作 | 鼠标悬停高亮资源，点击查看详情，支持多选 |
| IDM式并行下载 | 将文件分成多块并行下载，实时显示下载进度 |
| 多种导出方式 | 文件夹输出 / html演示文件 / Excel/CSV/MD/TXT 表格导出 |
| 任务流记录 | 自动记录每次抓取操作的时间、URL、资源数量与耗时 |
| HT 编辑器 | 专业菜单栏（文件/编辑/插入/视图），22 种卡片类型，撤销/重做，多选对齐 |
| 统计图 | 柱状图/饼图/折线图/词云，Canvas 自绘，支持分箱/聚合，导出 HTML 可渲染 |
| 展览编辑器 | 自定义卡片框布局，4 种展示方式，方向性过渡，聚焦/弱化，概览面板 |
| AI 工作流 | crosspage 跨页抓取、template 末端抓取、tracking 追踪、链式运行、AI 辅助 |
| 抓取方案模板 | 内置招聘/评论/商品模板，用户可另存为自定义模板 |
| MCP 服务端 | 支持外部工具通过 MCP 协议调用应用功能 |
| 安全特性 | PIN 码保护、自动锁定、敏感数据加密存储、API Key 脱敏显示 |
| 多主题 | 白天/黑夜模式，CSS 变量 + 0.3s 过渡 |

## 下载安装

> **v2.0.1 起，Windows 与 Linux 安装包分开构建和分发**，用户只需下载对应平台的安装包，无需拉取全部平台产物。

### Windows

- **安装版**：下载 `Web.Scout-2.0.1-Setup.exe`，双击安装，支持选择安装目录
- **便携版**：下载 `Web.Scout-2.0.1-portable.exe`，免安装双击运行

> Windows 11 首次运行会有 SmartScreen 警告，点击「仍要运行」即可（应用未签名）。

### Linux

- **deb 包**：下载 `Web.Scout-2.0.1.deb`，执行 `sudo dpkg -i Web\ Scout-2.0.1.deb` 安装

适用 Debian 10+ / Ubuntu 18.04+ / 其他基于 deb 的发行版。

### 从 Releases 获取

访问 [Releases 页面](https://github.com/HSyilin/web-scout/releases)，选择对应版本，只下载所需平台的安装包即可。

### 从源码运行

```bash
# 环境要求：Node.js 18+
git clone https://github.com/HSyilin/web-scout
cd web-scout
npm install
npm start
```

### 构建安装包

```bash
# Windows（需在 Windows 上执行）
npm run build:win

# Linux（需在 Linux 上执行，或在 Windows 上安装 wine）
npm run build:linux
```

## 使用流程

### 1. 基础抓取流程

1. **启动应用** → 输入网址浏览网页
2. **开启抓取模式** → 点击右上角「抓取模式」开关
3. **自动扫描** → 页面资源自动分类显示（图片/视频/音频/链接/文本）
4. **选择资源** → 点击资源卡片查看详情，勾选需要的资源
5. **导出数据** → 支持文件夹输出、WSW演示文件、Excel/CSV/MD/TXT 表格

### 2. 使用抓取方案模板

1. 进入「AI 工作流」模块
2. 点击「➕ 新建任务」→ 选择任务类型（template 末端抓取 / crosspage 跨页抓取）
3. Step 1 顶部点击「📂 导入抓取方案模板」
4. 选择内置模板（如「嵌入式工程师招聘」）或用户自定义模板
5. 模板自动填入 sampleUrls / fields / urls / 任务名
6. Step 2 可自由修改字段（增删改）或通过可视化拾取添加新字段
7. Step 3 填入目标 URL（多个）
8. Step 4 设置存放地址（必选）和自动导出选项，保存任务
9. 点击「▶ 运行」执行任务，结果自动保存到指定目录
10. 可选：Step 4 点击「💾 另存为模板」把当前配置保存为用户模板供下次复用

### 3. HT 编辑器数据可视化（v2.0.1 增强）

1. 进入「HT 编辑器」模块，新建文档
2. 使用专业菜单栏（文件/编辑/插入/视图）添加卡片：
   - **文字/表格/图表**：基础数据展示
   - **数据卡片**（新增）：指标卡 / 仪表盘 / 统计卡 / 对比卡 / 漏斗图 / 流程图
   - **辅助卡片**：注解 / 分隔线 / 二维码 / 时间轴 / 雷达图
3. 多选卡片后用「对齐」工具进行 8 种对齐（左/右/顶/底/水平居中/垂直居中/水平等距/垂直等距）
4. 操作过程中可用「撤销/重做」按钮（↶/↷）回退
5. 卡片内容超出时自动出现滚动条，不再被截断
6. 点击「保存为 HTML」导出自包含 HTML

### 4. 展览编辑器（v2.0.1 新增）

1. 在 HT 编辑器中点击「✎ 展览编辑」进入展览编辑器
2. 首次进入自动从当前卡片生成分页配置（可在此基础上调整）
3. **添加卡片框**：点击「▭ 添加框」创建新卡片框
4. **调整框**：拖动框移动位置，拖右下角手柄调整大小
5. **放入卡片**：双击卡片框，在弹出的卡片选择器中选择卡片（带内容预览）
6. **选展示方式**：选中框后选择展示方式（适应/填充/拉伸/居中）
7. **翻页/加页**：可编辑多页，每页独立的卡片框布局
8. 点击「▶ 播放」进入展览播放，用 ←/→ 翻页，ESC 退出
9. 播放时支持方向性过渡、聚焦/弱化、概览面板、侧边导航点

### 5. 多数据抓取与分析

1. 打开AI工作流
2. 根据提示选择合适的抓取流卡片
3. 可视化辅助多选抓取

### 6. 懒人专用

1. 内置MCP服务，打开任意支持MCP服务的智能体应用(trea work\workbuddy\...)
2. 将项目mcp添加到智能体应用上，或者将项目放在对应工作空间下
3. 告诉它你要抓取的内容和导出的方式

## 技术栈

- **Electron 28** - 桌面应用框架
- **原生 HTML/CSS/JS** - 轻量级界面，无前端框架依赖
- **XLSX** - Excel 文件读写
- **PDFKit** - PDF 文档生成
- **docx** - Word 文档生成
- **@modelcontextprotocol/sdk** - MCP 协议支持

## CI/CD

Windows 与 Linux 安装包**分别独立构建**，避免用户拉取全部平台产物：

- **Windows 构建**（[build-windows.yml](.github/workflows/build-windows.yml)）：在 `windows-latest` 上运行，产出 NSIS 安装包和便携版 exe，上传到 GitHub Releases
- **Linux 构建**（[build-linux.yml](.github/workflows/build-linux.yml)）：在 `ubuntu-22.04` 上运行，产出 deb 包，上传到 GitHub Releases

**触发方式**：
- 打 tag `v*` 自动触发两个平台的并行构建
- 也可在 GitHub Actions 页面手动触发（workflow_dispatch）

**用户获取安装包**：访问 [Releases](https://github.com/HSyilin/web-scout/releases) 页面，只下载所需平台的安装包。

## 开发日志


### 阶段一：基础浏览器框架
- 使用 TRAE 搭建 Electron 主进程 + 渲染进程架构
- 实现多标签页浏览、地址栏导航、前进后退

### 阶段二：资源扫描引擎
- 通过 `webview-preload.js` 注入页面，自动识别图片/视频/音频/链接/文本

### 阶段三：IDM式并行下载
- 实现 HTTP Range 分块并行下载
- 实时进度条显示（已下载MB / 总大小MB）

### 阶段四：多格式导出
- 文件夹输出：按类型分类保存
- WSW演示文件：网格布局展示资源
- Excel/CSV/MD/TXT 表格：多媒体按行列输出

### 阶段五：HT 编辑器与 AI 工作流
- Qt5 风格画布，多种卡片类型
- AI 工作流：crosspage/template/tracking 任务类型，链式运行
- MCP 服务端，支持外部工具调用

### 阶段六：v2.0.0 升级
- HT 编辑器统计图增强：CSV/JSON/XLSX 导入、分组聚合、薪资分箱、导出 HTML 渲染
- 抓取方案模板系统：6 个内置模板 + 用户模板 CRUD

### 阶段七：v2.0.1 专业化升级
