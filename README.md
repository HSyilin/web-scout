# Web Scout - 可视化智能爬虫

一款轻量级、跨平台的桌面应用，让网页资源获取像浏览一样简单。

## 功能特点

- **双模式切换**：默认浏览模式，一键开启抓取模式
- **资源自动识别**：扫描页面中的图片、视频、音频、PDF文档和文本内容
- **可视化操作**：鼠标悬停高亮资源，点击查看详情
- **三种导出方式**：
  1. 文件夹输出 - 按类型分类保存到指定目录
  2. WSW演示文件 - 网格布局展示资源，像PPT一样演示
  3. Excel表格导出 - 多媒体按行列输出，文本可合并编号填充
- **任务流记录**：自动记录每次抓取操作，形成结构化数据

## 安装与运行

### 前置要求

- Node.js 18+ 已安装
- npm 或 yarn 已安装

### 安装步骤

```bash
# 1. 进入项目目录
cd web-scout

# 2. 安装依赖
npm install

# 3. 运行应用
npm start
```

### 开发模式

```bash
# 运行应用并打开DevTools
npm start -- --dev
```

## 构建发布版本

```bash
# 构建 Windows 版本
npm run build:win

# 构建 Linux 版本
npm run build:linux

# 构建所有平台
npm run build
```

构建产物位于 `dist/` 目录。

## 使用说明

1. **启动应用**：打开后显示空白页，输入网址开始浏览
2. **开启抓取模式**：点击右上角「抓取模式」开关
3. **扫描资源**：抓取模式下自动扫描页面资源
4. **选择资源**：点击资源查看详情，可多选
5. **导出数据**：
   - 点击「批量下载」导出到文件夹
   - 点击「创建 WSW 演示」生成演示文件
   - 点击「导出 Excel」生成表格文件

## 项目结构

```
│   │   └── index.js    # 主进程入口

│   │   └── index.js    # 暴露安全API

├── assets/             # 应用图标等资源
├── package.json        # 项目配置
└ └── README.md         # 说明文档
```
