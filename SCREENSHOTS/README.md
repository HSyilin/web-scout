# 截图说明

本目录用于存放比赛所需的截图材料。

## 需要的截图

### 1. 产品功能截图（3-5张）

| 文件名 | 内容说明 |
|--------|----------|
| `01-main-interface.png` | 主界面浏览模式（地址栏、标签页、侧边栏） |
| `02-capture-mode.png` | 抓取模式开启后，资源自动分类显示 |
| `03-resource-selection.png` | 资源卡片选择界面，显示已选资源 |
| `04-parallel-download.png` | IDM式并行下载进度条显示 |
| `05-export-options.png` | 三种导出方式选择界面 |

### 2. TRAE 开发过程截图（至少3张）

| 文件名 | 内容说明 |
|--------|----------|
| `dev-01-browser-framework.png` | TRAE 对话：搭建浏览器框架 |
| `dev-02-resource-scanner.png` | TRAE 对话：资源扫描引擎开发 |
| `dev-03-parallel-download.png` | TRAE 对话：并行下载逻辑实现 |
| `dev-04-export-feature.png` | TRAE 对话：导出功能开发 |

## 截图要求

1. **清晰度：** 截图分辨率不低于 1920x1080
2. **完整性：** 包含完整的窗口界面，不要裁剪
3. **TRAE 对话截图：** 需要显示 Session ID（双击对话头像可复制）
4. **格式：** PNG 或 JPG 格式

## 如何获取 Session ID

1. 在 TRAE IDE 中打开对话
2. 双击对话头像
3. 复制弹出的 Session ID
4. 填写到 DEMO_POST.md 中对应位置

## 截图后下一步

1. 将截图放入本目录
2. 更新 DEMO_POST.md 中的截图引用
3. 在 TRAE 社区发帖时上传这些截图
